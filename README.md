## Overview
Deal Detective is a software design project focused on the planning and specification of a price comparison web application.
The goal is to help users compare product prices across multiple online retailers and identify the best available deals.

## Problem Statement
Online shoppers often need to manually browse multiple platforms to compare prices.
Deal Detective aims to centralize product information and pricing to improve user decision-making.

## Core Features (Proposed)
- Product search and comparison
- Price aggregation from multiple retailers
- User accounts and wishlists
- Category-based filtering
- Scalable system architecture

## System Design
The project includes:
- Functional and non-functional requirements
- High-level system architecture
- Database design (relational schema)
- User workflow and use-case analysis

## Proposed Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## Project Type
This was a **software design and requirements engineering project**.
No production code was implemented.
